# Finance Analytics & Stock Portfolio Dashboard (2025-26)

A high-fidelity, interactive Power BI dashboard designed to provide comprehensive financial insights and stock market performance tracking. This project focuses on modern data visualization techniques and advanced DAX logic to simplify complex financial datasets.

## 📊 Dashboard Highlights

* **Executive Summary:** Real-time tracking of Core KPIs including **Revenue ($11M)**, **Net Income ($3M)**, **Gross Profit ($6M)**, and **EBITDA ($4M)**.
* **Stock Portfolio Management:** Detailed breakdown of market value by individual holdings (Nvidia, Microsoft, Apple, etc.) and sector allocation using Treemaps and Donut charts.
* **P&L Analysis:** A monthly Waterfall chart depicting profit and loss movements, alongside a Revenue vs. Net Income trend analysis.
* **Interactive Slicers:** Full dynamic filtering by **Year**, **Sector**, **Stock**, and **Quarter**.
